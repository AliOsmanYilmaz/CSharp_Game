# 🎮 C# Console Applications & Games

This repository contains a collection of fun console-based applications and game examples developed using C#. It reflects my learning journey, focusing on core programming logic and algorithmic structures.

## 🚀 Key Features
The current version of the project includes:
* **User Interaction:** Dynamic data input through the console.
* **Game Logic:** Implementation of basic loops, conditional statements, and logic flow.
.

## 🛠️ Installation & Setup

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/AliOsmanYilmaz/CSharp_Game.git](https://github.com/AliOsmanYilmaz/CSharp_Game.git)
    ```
2.  **Open the project:**
    Locate and open the `.sln` file using Visual Studio or VS Code.
3.  **Run the application:**
    Press `F5` or click the `Start` button to compile and run the project.
